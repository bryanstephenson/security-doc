========================
OpenStack Security Guide
========================

Abstract
~~~~~~~~

This book provides best practices and conceptual information about
securing an OpenStack cloud.

This guide documents OpenStack Mitaka, OpenStack Liberty, and OpenStack
Kilo releases.
